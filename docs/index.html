[8 教學反思與省思]

這些年來，不斷的在教學方法及內容上嘗試改變，三不五時也會覺得非常非常的疲憊，有時覺得自己做得還不夠好，偶爾則會抱怨一下學生的漫不經心，就像我老婆常虧我的那句話「甘願作，歡喜受」，我還是會持續的動態調整教學策略，希望能慢慢的變成「更適合」南臺學生的學習導師。
