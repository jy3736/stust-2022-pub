### [問題 7]：依指定資料寬度及每一行(row)的列數(col)輸出一維陣列

- 完成主程式中使用的劃線函數 `dump(...)`。
- 函數 `dump(...)`有三種不同的用，你可選擇以下兩種設計：
    - 內定參數的方式設計單一函數。
    - 設計3個覆載(overloading)函數。
